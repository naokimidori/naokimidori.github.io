前言

在前端开发中，经常会遇到频繁触发某一事件的情况，如 scroll、mousemove、onchange 等。这种高频率的触发会造成卡顿等现象。
解决这种问题通常有两种方法：防抖 和 节流

防抖 debounce

防抖原理

在事件被触发 n 秒后再执行回调，如果在这 n 秒内又被触发，则重新计时。
